# Интеграция 1С с сервисом разбора заявок

## Архитектура

```
1С Конфигурация
├── Общий модуль: РазборЗаявокAPI (Клиент)  ← вся HTTP-логика
│   ├── РазобратьТекст(текст) → Структура
│   ├── РазобратьФайл(путь) → Структура (JSON + Base64)
│   └── ПроверитьСоединение() → Структура
└── Обработка: МастерОбработкиЗаявки  ← пошаговый визард
    ├── Этап 1: выбор источника (текст МП / файл)
    └── Этап 2: результат разбора + подбор номенклатуры
```

---

## Общий модуль: РазборЗаявокAPI

**Свойства:**
- [x] Клиент (управляемое приложение)
- [ ] Сервер — **НЕ включать** (служба 1С блокирует HTTP-подключения)
- [ ] Вызов сервера — не нужен

**Файл:** `CommonModule_РазборЗаявокAPI.bsl`

**Настройки подключения** зашиты в модуле (по умолчанию `127.0.0.1:8000`).
Если сервис на другой машине — поменять `Адрес` и `Порт` в `ПолучитьНастройкиПодключения()`.

## Обработка: Мастер обработки заявки

### Реквизиты формы:

| Имя                   | Тип          | Назначение                        |
|-----------------------|--------------|-----------------------------------|
| ТекущийЭтап           | Число(1,0)   | 0 = выбор источника, 1 = результат|
| ИсточникДанных        | Число(1,0)   | 0 = текст от МП, 1 = файл        |
| ТекстОтМП             | Строка(0)    | Многострочный текст сообщения     |
| ПутьКФайлу            | Строка(500)  | Путь к файлу Excel/CSV            |
| ТекстСтатистики       | Строка(200)  | Сводная строка (только чтение)    |
| ТекстПредупреждений   | Строка(0)    | Список пропущенных полей          |

### Табличная часть «Позиции»:

| Имя колонки       | Тип          | Назначение                      |
|--------------------|--------------|---------------------------------|
| НомерСтроки        | Число(5,0)   | Порядковый номер                |
| Наименование       | Строка(500)  | Исходный текст позиции          |
| Количество         | Число(15,3)  | Распознанное количество         |
| ЕдИзмерения        | Строка(20)   | Единица измерения               |
| ТипИзделия         | Строка(100)  | Тип (болт, гайка...)            |
| Размер             | Строка(50)   | Размер (М12х80)                 |
| Диаметр            | Строка(50)   | Диаметр (если отдельно)         |
| Стандарт           | Строка(100)  | ГОСТ / DIN / ISO                |
| Прочность          | Строка(50)   | Класс прочности                 |
| Покрытие           | Строка(100)  | Покрытие                        |
| Статус             | Строка(20)   | ok / review / manual            |
| НеЗаполнено        | Строка(500)  | Что не распознано               |
| НашаНоменклатура   | Строка(500)  | Подобранная позиция из каталога |
| УИД_1С             | Строка(36)   | УИД номенклатуры                |
| РежимПодбора       | Строка(30)   | авто / предложено / не найдено  |
| Совпадение         | Число(5,1)   | Процент совпадения (0-100)      |

### Группы формы:

| Имя группы              | Назначение                                |
|-------------------------|-------------------------------------------|
| СтраницаВыборИсточника  | Всё содержимое этапа 1                    |
| СтраницаРезультат       | Всё содержимое этапа 2                    |
| ГруппаТекстМП           | Поле ТекстОтМП (видима при Источник = 0)  |
| ГруппаФайл              | ПутьКФайлу + кнопка (видима при Источник=1)|
| ГруппаКнопкиЭтап1       | Кнопка «Далее», «Проверить связь»         |
| ГруппаКнопкиЭтап2       | Кнопка «Назад»                            |

### Команды формы:

| Команда         | Действие                                    |
|-----------------|---------------------------------------------|
| Далее           | Отправить данные на разбор → перейти к Этап 2|
| Назад           | Вернуться к Этап 1                           |
| ВыбратьФайл     | Диалог выбора файла                          |
| ПроверитьСвязь  | Тест подключения к API                       |

**Файл модуля формы:** `wizard_form.bsl`

## Вызов из других мест 1С

```1c
// Из любого клиентского модуля:
Результат = РазборЗаявокAPI.РазобратьТекст("Болт М12х80 100 шт");
Если Результат.Успех Тогда
    Для Каждого Поз Из Результат.Позиции Цикл
        Сообщить(Поз.Наименование + " → " + Поз.НашаНоменклатура);
    КонецЦикла;
КонецЕсли;

// Файл:
Результат = РазборЗаявокAPI.РазобратьФайл("C:\заявка.xlsx");
```
