# Интеграция 1С с сервисом разбора заявок

## Архитектура

```
1С Конфигурация
├── Константы (настройки подключения)
├── Общий модуль: РазборЗаявокAPI  ← вся HTTP-логика
│   ├── РазобратьТекст(текст) → Структура
│   ├── РазобратьФайл(путь) → Структура
│   ├── РазобратьДвоичныеДанные(данные, имя) → Структура
│   └── ПроверитьСоединение() → Структура
└── Обработка: РазборЗаявкиКлиента  ← UI-форма
```

Общий модуль можно вызывать откуда угодно:
из обработок, документов, регламентных заданий.

---

## Шаг 1. Создать константы

В Конфигураторе → Общие → Константы, создать:

| Имя                             | Тип       | Значение по умолчанию |
|---------------------------------|-----------|-----------------------|
| РазборЗаявок_АдресСервера       | Строка(200) | localhost           |
| РазборЗаявок_ПортСервера        | Число(5,0)  | 8000                |
| РазборЗаявок_Таймаут            | Число(3,0)  | 60                  |

Если константы не созданы — модуль использует значения по умолчанию.

---

## Шаг 2. Создать общий модуль

1. Конфигуратор → Общие → Общие модули → Добавить
2. Имя: `РазборЗаявокAPI`
3. Свойства:
   - [x] Сервер
   - [x] Вызов сервера
   - [ ] Клиент (не нужен)
   - [ ] Внешнее соединение (по желанию)
4. Скопировать код из файла `CommonModule_РазборЗаявокAPI.bsl`

---

## Шаг 3. Создать обработку

1. Конфигуратор → Обработки → Добавить
2. Имя: `РазборЗаявкиКлиента`

### Реквизиты обработки:

| Имя               | Тип          |
|--------------------|-------------|
| ТипВвода           | Число(1,0)  |
| ТекстЗаявки       | Строка(0)   |
| ПутьКФайлу         | Строка(500) |
| ТекстСтатистики    | Строка(200) |

### Табличная часть «Позиции»:

| Имя колонки       | Тип          | Назначение                      |
|--------------------|--------------|---------------------------------|
| НомерСтроки        | Число(5,0)   | Порядковый номер                |
| Наименование       | Строка(500)  | Исходный текст позиции          |
| Количество         | Число(15,3)  | Распознанное количество         |
| ЕдИзмерения        | Строка(20)   | Единица измерения               |
| ТипИзделия         | Строка(100)  | Тип (болт, гайка...)            |
| Размер             | Строка(50)   | Размер (М12х80)                 |
| Диаметр            | Строка(50)   | Диаметр (если отдельно)         |
| Стандарт           | Строка(100)  | ГОСТ / DIN / ISO                |
| Прочность          | Строка(50)   | Класс прочности                 |
| Покрытие           | Строка(100)  | Покрытие                        |
| Статус             | Строка(20)   | ok / review / manual            |
| НеЗаполнено        | Строка(500)  | Что не распознано               |
| НашаНоменклатура   | Строка(500)  | Подобранная позиция из каталога |
| УИД_1С             | Строка(36)   | УИД номенклатуры                |
| РежимПодбора       | Строка(30)   | авто / предложено / не найдено  |
| Совпадение         | Число(5,1)   | Процент совпадения (0-100)      |

### Форма обработки:

Создать управляемую форму, разместить элементы:

```
┌─────────────────────────────────────────────────────┐
│  [Проверить связь]                                   │
│                                                       │
│  Тип ввода: (o) Текст   ( ) Файл Excel/CSV          │
│                                                       │
│  ┌─ Группа Текст ─────────────────────────────────┐  │
│  │  [ Многострочное поле ТекстЗаявки            ] │  │
│  │  [ ________________________________          ] │  │
│  └────────────────────────────────────────────────┘  │
│                                                       │
│  ┌─ Группа Файл (скрыта при ТипВвода=0) ─────────┐  │
│  │  ПутьКФайлу: [_______________] [Выбрать файл]  │  │
│  └────────────────────────────────────────────────┘  │
│                                                       │
│  [ === Разобрать заявку === ]  [Очистить]            │
│                                                       │
│  Статистика: Всего: 5 | OK: 3 | На проверку: 1 ...  │
│                                                       │
│  ┌─ Таблица Позиции ─────────────────────────────┐  │
│  │ # │ Наименование │ Кол │ Размер │ Статус │ ...│  │
│  │ 1 │ Болт М12х80  │ 100 │ М12х80 │ ok     │    │  │
│  │ 2 │ Гайка М16    │  50 │ М16    │ review │    │  │
│  └────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

### Команды формы:
- `РазобратьЗаявку` — главная кнопка
- `ПроверитьСвязь` — тест подключения к API
- `ВыбратьФайл` — диалог выбора файла
- `Очистить` — очистка таблицы

### Группы:
- `ГруппаТекст` — содержит ТекстЗаявки (видима при ТипВвода = 0)
- `ГруппаФайл` — содержит ПутьКФайлу + кнопку (видима при ТипВвода = 1)

4. Скопировать код из файла `module_form.bsl` в модуль формы

---

## Шаг 4. Проверить

1. Запустить сервис: `python -m uvicorn app.main:app --host 0.0.0.0 --port 8000`
2. В 1С открыть обработку
3. Нажать «Проверить связь» — должно показать «Сервис доступен»
4. Ввести текст:
   ```
   Болт М12х80 ГОСТ 7798-70 8.8 цинк 100 шт
   Гайка М16 DIN 934 50 шт
   Шайба 20 ГОСТ 11371
   ```
5. Нажать «Разобрать заявку»

## Использование из других мест 1С

```1c
// Из любого модуля:
Результат = РазборЗаявокAPI.РазобратьТекст("Болт М12х80 100 шт");
Если Результат.Успех Тогда
    Для Каждого Поз Из Результат.Позиции Цикл
        Сообщить(Поз.Наименование + " → " + Поз.НашаНоменклатура);
    КонецЦикла;
КонецЕсли;

// Из документа (например, при загрузке заявки клиента):
Результат = РазборЗаявокAPI.РазобратьФайл("C:\заявка.xlsx");

// Можно передать двоичные данные напрямую (из вложения письма и т.п.):
Результат = РазборЗаявокAPI.РазобратьДвоичныеДанные(ДвоичныеДанные, "заявка.xlsx");
```
