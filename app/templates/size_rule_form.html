Именованные группы в regex: (?P<d>...) — диаметр, (?P<l>...) — длина, (?P<w>...) — ширина, (?P<t>...) — толщина, (?P<tol>...) — допуск.
Шаблон нормализации: M{d}x{l}, {d}x{t}x{w}.

Регулярное выражение. Текст предварительно: М→M, х→x, *→x, запятая→точка.

Необязательно. Python format string с именами групп: {d}, {l}, {w}, {t}, {tol}.

Больше = выше приоритет. Первое сработавшее правило побеждает.
