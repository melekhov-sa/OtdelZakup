Загрузить JSON из 1С

Файл групп номенклатуры
Файл номенклатуры
Что происходит при загрузке:
  • Все папки из файла добавляются или обновляются (приоритеты сохраняются)
  • Для каждой позиции: из наименования + характеристики автоматически извлекаются тип, размер, стандарт, прочность, покрытие
  • Существующие позиции обновляются; новые добавляются
  • Позиции 1С, отсутствующие в файле, деактивируются (не удаляются)
  • MinHash-индекс пересобирается автоматически

Ожидаемый формат файлов

Файл групп (массив)
[ { "folder_uid": "a1b2c3d4-...", "folder_name": "Крепёж основной", "parent_uid": null, "folder_path": "Крепёж основной" }, ... ]
Файл номенклатуры (массив)
[ { "uid_1c": "d4e5f6g7-...", "uid_1c_char": null, "name": "Болт М8х50 ГОСТ 7798-70", "char_name": "Цинк · 5.6", "folder_uid": "a1b2c3d4-...", "folder_name": "Крепёж основной", "folder_path": "Крепёж основной", "is_active": true }, ... ]
