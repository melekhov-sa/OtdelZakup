Правила вывода размера

← Главная

Определяют, как вычислить поле Размер, если оно не указано явно в тексте.
Для гайки и шайбы размер совпадает с диаметром (например, M20).
Для болта, винта, анкера размер формируется как диаметр × длина (например, M12x150).
Правила применяются по приоритету (меньше = раньше). Если правило сработало — размер считается заполненным для проверки готовности.
