Для использования Google Document AI необходим GCP-проект с включённым API Document AI и созданным процессором.
Аутентификация: по умолчанию используются Application Default Credentials — JSON ключ не обязателен. Для локальной работы выполните:
gcloud auth application-default login
JSON сервисного аккаунта нужен только если ADC недоступен.

Подключение

ID вашего Google Cloud проекта (не название, а идентификатор).
Регион, где создан процессор: eu или us.
ID процессора Document AI из GCP Console → Document AI → Мои процессоры.
